# Dotfiles

## Setup

- Terminal: alacritty
- WM: sway
- Shell: ZSH - OMZ
- Bar: i3status-rs
- Menu: dmenu
- Editor: neovim

## Pacman packages
These packages are recommended to install when installing Arch:

```bash
sudo pacman -Syu \
networkmanager gcc git ttf-dejavu ttf-fira-code \
wl-copy \
slurp \
wl-clipboard \
grim
```
