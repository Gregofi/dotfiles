# Dotfiles

## Setup

- Terminal: allacrity
- WM: i3
- Shell: ZSH - Prezto
- Bar: i3status-rs
- Menu: dmenu
- Editor: neovim

## Neovim
The telescope requires `ripgrep` package to work.

## Pacman packages
These packages are recommended to install when installing Arch:
```
networkmanager gcc git ttf-dejavu ttf-liberation noto-fonts
```
